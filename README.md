# raspberry-server
Remote internet of things centralized server

### Development tools
- [x] SSH and VNC access
- [x] Remote python interpreter and debugger - JetBrains Pycharm & ArchLinux

### Utilities
- [x] Wifi router and hotspot
- [x] Communication with embedded IoT devices
- [x] Web server
- [x] Databases
